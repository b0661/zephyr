Description
***********

The Chaos Preprocessor Library is a generative metaprogramming framework for C and C++ created by Paul Mensonides. It is a header-only library.

Concepts
********

Paul Mensonides:

  "Chaos" does mean something...  It is a reference to chaos theory where a relatively simple set of rules
  and initial conditions can produce a complex result.
